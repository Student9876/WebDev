Image Insert <img>

With this tag, we can insert image in out HTML file

Attributes

  • src: The Image URL In this tag, there can be an address of an image. The address can be local address or web address
  • width: The intrinsic width of the image in pixels. Must be an integer without a unit.
  • heigth: The intrinsic height of the image, in pixels. Must be an integer without a unit.
  • alt: Defines an alternative text description of the image
  • border: The width of a border around the image. Can be used border in CSS properties
  • align: Aligns the image with its surrounding context
    Allowed values:
    1. top: Equivalent to vertical-align: top or vertical-align: text-top
    2. middle: Equivalent to vertical-align: -moz-middle-with-baseline
    3. bottom: The default, equivalent to vertical-align: unset or vertical-align: initial
    4. left: Equivalent to float: left
    5. right: Equivalent to float: right
  • decoding: Provides an image decoding hint to the browser. Allowed values:
    1. sync: Decode the image synchronously, for atomic presentation with other content.
    2. async: Decode the image asynchronously, to reduce delay in presenting other content.

Anchor Element<a>

The <a> HTML element, with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address

Attributes

  • href: The URL that the hyperlink points to. Links are not restricted to HTTP-based URLs--They can use any URL scheme supported by browsers:
    1. Sections of page with fragment URLs
    2. Pieces of media files with media fragments
    3. Telephone numbers with tel: URLs
    4. Email addresses with mailto:
  • ping: A space-separated list of URLs. When the link is followed, the browser will send POST requests with the body PING to the URLs. Typically for tracking
  • type: Hints at the linked URL's format with a MIME type. no built-in functionality

Table<table>

This tag is used to create table in HTML. It has various attributes

Attributes

  • align
    1. left: the table is displayed on the left side of the document
    2. center: the table is displayed on the center of the document
    3. left: the table is displayed on the right side of the document
  • bgcolor: The background color of the table. It's a 6 digit hexadecimal RGB code, prefixed by a #
  • border: This integer attribute defines, in pixels, the size of the frame surrounding the table.
  • summeryThis attribute defines an alternative text that summarizes the content of the table.
  • cellspacing: This attribute do the size adjustments of the space between two cells in a percentage value or pixels. This attribute is applied both horizontally and vertically

Form Element <form>

This HTML element provides a document section which contain interactive controls for submitting Information

Attributes

  • name: The name of the form. The value must not be the empty string, and must be unique among the form elements in the forms collection that it is in.
  • action: It does some action when the submit key is pressed.
    e.g -    <form action="index.html"> can take you back to the index.html page of your website, <form action="mailto:(email_address)"> can take you to the email after pressing the submit button.
  • enctype: If the value of the method attribute is post, enctype is the MIME (Multipurpose Internet Mail Extensions) type of the form submission.
    Types of enctype:
    1. application/x-www-form-urlencoded: It is the default value
    2. text/plain: It prints the input values as plain text.
    3. multipart/form-date: Use this if the form contains <input> elements with the type=file

Methods

  1. <label>: It is used to print some text.
  2. <input>: This tag is used to take input from the user.

    <input> attributes:

    • type:
      Different types
      • text: This attribute is used inside label to take text inputs.
      • email: This attribute is used to take input email addresses
      • submit: Using this attribute a button appears. With it we can perform submit or whatever task we program.
      • password: With this tag we can input password. This tag takes a masked input to avoid the password get leaked.
      • checkbox: This gives us a checkbox to check it.
      • radio: This gives us a radio button to select some option.
      • color: It takes colors as input.
      • date: This attribute gives us a calender to enter any date.
